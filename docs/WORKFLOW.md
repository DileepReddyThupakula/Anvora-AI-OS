# Anvora AI OS Workflow

1. Research Agent
   - Collect AI news
   - Rank stories
   - Save research report

2. Script Agent
   - Read research report
   - Generate YouTube script
   - Save script

3. Voice Agent
   - Generate narration
   - Export audio

4. Video Agent
   - Match visuals
   - Add subtitles
   - Render video

5. Thumbnail Agent
   - Generate thumbnail prompt
   - Produce thumbnail

6. Publisher Agent
   - Generate title
   - Generate description
   - Generate tags
   - Prepare upload

7. Analytics Agent
   - Read YouTube statistics
   - Recommend improvements
